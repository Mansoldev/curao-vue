A parte del CLI, VUE cuenta con una serie de librerias necesarias para proyectos medianos y grandes:

  • Estructura de carpetas senior: Repaso a la estructura de carpetas y ampliación para proyectos smartlabs.
  • Vuetify: Es un framework CSS que nos brinda una gran lista de componentes estéticos y visuales.
  • Bus: Es una abstracción plugin para manejar el estado de la aplicación de forma simple.
  • Vue-router: Es una libreria que actua de enrutador de la aplicación desde el cliente, creando diferentes "páginas en el cliente".
  • vue-auto-routing: Routea automáticamente los componentes de una carpeta concreta como páginas unidas a una url.
  • vue-json-excel: libreria que exporta json a excel
